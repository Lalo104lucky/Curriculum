## Descripción del sistema

Este proyecto es una plataforma web sencilla que muestra los perfiles  de los integrantes del equipo. Cada perfil incluye información personal, intereses y hobbies, presentados de manera visual y atractiva. El objetivo es practicar el desarrollo web y la colaboración en equipo, utilizando HTML, CSS y frameworks como Bootstrap y Flowbite.

## Cómo clonar y correr el proyecto

1. **Clona el repositorio:**

   ```sh
   git clone https://github.com/tu-usuario/Curriculum.git
   ```

2. **Entra a la carpeta del proyecto:**

   ```sh
   cd Curriculum
   ```

3. **Abre el archivo index.html en tu navegador favorito.**

   ```sh
   Puedes hacer doble clic en el archivo index.html o abrirlo desde tu editor de código.
   ```
